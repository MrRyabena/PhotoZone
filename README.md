**PhotoZone** — небольшой проект, с помощью которого можно организовать красивое оформление банера/плаката/арт-объекта.</br>
[Видео с созданием.](https://youtu.be/TXuJKdMZ7J0)

В технической основе лежат RGB лента и прожектор, сделанный самостоятельно из большой светодиодной матрицы (проект Гайвера [RGBspot](https://alexgyver.ru/rgbspot/), но прошивка полностью переписана. Позже будет сделана версия прожектора с расширенным функционалом, приложением и объединением в сеть по WiFi).</br>

Прошивка написана на коленке чтобы работало здесь и сейчас.

- [strip](/src/strip/) — главный контроллер, отвечает за RGB-эффекты, опрос кнопки, управление лентой и синхронизацию цветов с прожектором.
- [spot](/src/spot/) — контроллер прожектора. В этом проекте управляет вентилятором охлаждения и выводит цвет, считанных с сигналов главного контроллера.

Эффекты переключаются по нажатию на сенсорную (или обычную) кнопку. Их не очень много, но, конечно, можно добавить сколько угодно:

- Статические цвета (красный, зеленый, синий, фиолетовый).
- Плавная смена цветов от красного до фиолетового и назад.
- Плавная смена цветов по полному цветовому колесу.
